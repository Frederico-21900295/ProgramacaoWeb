Todos os combinadores de seletores apresentados em aula. [6]
Selectores Usou? Como funciona? Onde usou?
Seletores descendentes: div p done Seleciona todos os ‹p› elementos dentro de ‹div› elementos. Usei por exemplo "section figure"
Seletores filhos: div > p done Seleciona todos os ‹p› elementos quando o "pai" é o ‹div› elemento. Usei por exemplo "section > table" nesta página.
Seletores adjacentes: div ~ p done Seleciona todos os ‹p› elementos que vêm a seguir ao ‹div› elemento. Usei por exemplo ".genero ~ label" na página do quizz.
Seletores imediatamente adjacentes: div + p done Seleciona o primeiro ‹p› elemento que está posicionado aseguir ao ‹div› elemento. Usei por exemplo "section + h3" na página do quizz.
Agrupar os seletores: div,p done Seleciona todos os ‹p› e os ‹div› existentes na página. Usei por exemplo "header, nav, .centrado" na página de multimédia..
